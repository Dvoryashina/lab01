# Contributing to Lab01

Спасибо за ваш интерес к проекту! Мы рады любому вкладу в развитие этого проекта.

## Как внести вклад

### Сообщение о проблемах

Если вы обнаружили ошибку или хотите предложить улучшение:

1. Проверьте, не была ли эта проблема уже описана в Issues
2. Создайте новый Issue с подробным описанием
3. Используйте понятный заголовок и описание
4. Приложите примеры кода или скриншоты, если возможно

### Процесс разработки

1. **Fork репозитория**
   ```bash
   gh repo fork Dvoryashina/lab01
   ```

2. **Создайте ветку для вашей функции**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Внесите изменения**
   - Следуйте существующему стилю кода
   - Добавьте комментарии к сложному коду
   - Обновите документацию при необходимости

4. **Закоммитьте изменения с GPG подписью**
   ```bash
   git add .
   git commit -S -m "Описание изменений"
   ```

5. **Отправьте изменения в ваш fork**
   ```bash
   git push origin feature/your-feature-name
   ```

6. **Создайте Pull Request**
   ```bash
   gh pr create --title "Краткое описание" --body "Подробное описание изменений"
   ```

## Стандарты кода

### Python

- Следуйте PEP 8
- Используйте понятные имена переменных
- Документируйте функции с помощью docstrings
- Максимальная длина строки: 88 символов (Black formatter)

### Git коммиты

- Используйте информативные сообщения коммитов
- Начинайте с глагола в повелительном наклонении
- Все коммиты должны быть подписаны GPG (-S флаг)
- Формат: `Тип: краткое описание`

Примеры:
```
Добавлен: новая функция форматирования
Исправлен: баг с обработкой пустых строк
Обновлен: README с инструкциями по установке
```

## Тестирование

Перед отправкой Pull Request убедитесь, что:

- Код работает корректно
- Приложение запускается без ошибок
- Все существующие функции продолжают работать

Запуск приложения:
```bash
source venv/bin/activate
python typersteel.py "Тест" --lastname "Тестович" --formal
```

## Проверка перед коммитом

```bash
# Проверка стиля кода
python -m py_compile typersteel.py

# Проверка типов
python typersteel.py --help
```
